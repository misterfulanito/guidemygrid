# Task 008: Integration testing — PS 22-25, macOS + Windows

## Descripción

Testing manual completo del plugin end-to-end en Photoshop. No hay testing automatizado posible para las APIs de PS — esta task es 100% manual. Documentar cualquier comportamiento inesperado.

## Test Plan

### 1. Flujo Principal — Column Grid

| Test | Pasos | Resultado esperado |
|------|-------|-------------------|
| Aplicar 12 columnas auto | Config: 12 cols, 20px gutter, 80px márgenes. Clic "Reemplazar guías" | 24 guías verticales en posiciones matemáticamente correctas |
| Aplicar columnas fijas | Config: 6 cols, ancho fijo 100px, 10px gutter | 12 guías verticales, cada columna exactamente 100px |
| Márgenes respetados | Config: 12 cols, 80px margen L/R. Doc 1920px | Primer guía en x=80, última en x=1840 |
| Modo "Agregar" | Aplicar 6 cols, luego 3 cols en modo "Agregar" | Las guías se suman (no se borran las anteriores) |
| Modo "Reemplazar" | Aplicar 6 cols, luego 3 cols en modo "Reemplazar" | Solo quedan las 6 guías de 3 columnas |
| Sin documento | Abrir PS sin documento | Botón Apply deshabilitado, mensaje "Abre un documento" |
| Error márgenes grandes | Márgenes L+R > ancho documento | Error inline en el panel antes de intentar aplicar |
| Cambiar documento | Aplicar grid, cambiar a otro documento | `useDocument` actualiza dimensiones en tiempo real |

### 2. Presets

| Test | Pasos | Resultado esperado |
|------|-------|-------------------|
| Presets built-in | Abrir plugin por primera vez | "12-col Web Standard" y "8pt Grid" visibles |
| Guardar preset | Configurar grid, guardar como "Mi Grid" | Aparece en la lista con botón ▶ y ✕ |
| Persistencia | Guardar preset, cerrar PS, reabrir | Preset sigue guardado |
| Aplicar preset | Clic ▶ en preset | Los valores en tab Grid se actualizan |
| Eliminar preset | Clic ✕ | Preset eliminado, built-ins no tienen ✕ |
| Límite Free | Guardar 3 presets de usuario | Al 4to intento, botón deshabilitado con ProBadge |

### 3. UpdateBanner

| Test | Pasos | Resultado esperado |
|------|-------|-------------------|
| Sin update | PLUGIN_VERSION igual a la última release | No aparece banner |
| Con update | Bajar PLUGIN_VERSION en código y recompilar | Banner "Nueva versión vX.Y.Z disponible" |
| Descargar | Clic en "Descargar" | Se abre el browser con la URL correcta |
| Cerrar banner | Clic ✕ | Banner desaparece hasta la próxima sesión |
| Sin red | Desconectar red, reabrir plugin | Panel carga normalmente sin banner (sin error) |
