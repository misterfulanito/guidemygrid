## ColumnGrid.tsx

Spec en `docs/reference/technical-decisions.md` → Sección 9.2.

Inputs a implementar:
1. **Toggle** — enable/disable sección completa. Conectar a `gridStore.setColumnConfig({ enabled })`
2. **Columnas** — `NumberInput`, min=1, max=isPro?24:12, step=1. Con `ProBadge` si Free
3. **Ancho de columna** — Radio "Auto" / "Fijo" + `NumberInput` (px) condicional
4. **Gutter** — `NumberInput`, min=0, max=200, suffix="px"
5. **Margen izquierdo** — `NumberInput`, min=0, max=500, suffix="px"
6. **Margen derecho** — `NumberInput`, min=0, max=500, suffix="px"
7. **Ancho calculado** — texto informativo: `columnWidth = calcularAncho()` en px (P1)

```tsx
// Cálculo del ancho mostrado en tiempo real
const calculatedWidth = useMemo(() => {
  if (!document || !config.columns.enabled) return null;
  const available = document.width - config.columns.marginLeft - config.columns.marginRight;
  if (available <= 0) return null;
  if (config.columns.columnWidth === 'fixed') return config.columns.columnWidthValue;
  const w = (available - config.columns.gutter * (config.columns.columns - 1)) / config.columns.columns;
  return Math.round(w * 100) / 100;
}, [config.columns, document]);
```
