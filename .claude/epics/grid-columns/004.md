# Task 004: Hooks — useDocument + useLicense

## Descripción

Implementar los dos hooks React que conectan el estado del plugin con Photoshop y el sistema de licencias. Estos hooks son usados por los componentes UI — deben estar listos antes de implementar cualquier componente.

## Archivos a crear

```
src/hooks/
├── useDocument.ts
└── useLicense.ts
```

## Implementación

### src/hooks/useDocument.ts — Copiar directo
Copiar de `docs/reference/technical-decisions.md` → Sección 8.3.

Expone:
- `document: DocumentInfo | null` — dimensiones del doc activo
- `loading: boolean` — mientras carga al inicio

El hook escucha eventos de PS (`select`, `open`, `close`) para refrescarse automáticamente cuando el usuario cambia de documento.

**Validar en PS:** Abrir dos documentos con diferentes tamaños → cambiar entre ellos → verificar que `document.width/height` actualiza.

### src/hooks/useLicense.ts — Simplificado para MVP
Copiar base de `docs/reference/technical-decisions.md` → Sección 11.1 pero simplificar para MVP:

```typescript
// src/hooks/useLicense.ts
import { LicenseStatus } from '../types';

// MVP: plugin es gratuito. Todo el mundo es Free tier.
// Post-MVP: implementar uxp.entitlement.checkStatus()
export function useLicense(): LicenseStatus {
  return {
    tier: 'free',
    isActive: true,
  };
}
```

**Nota:** El hook devuelve siempre 'free' para MVP porque el plugin es gratuito. Las interfaces `LicenseStatus` y los `gates` de `featureGates.ts` están definidos y listos — solo falta activar la verificación real cuando se implemente monetización.

### src/services/featureGates.ts (si no existe aún)
