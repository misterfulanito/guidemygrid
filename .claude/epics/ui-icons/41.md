# Issue 003: Icons for show/hide guides and delete guides

## Context

The show/hide guides toggle and the delete all guides action currently use text labels or no visual indicator. Adding dedicated icons makes the actions immediately recognizable without reading text, consistent with Photoshop's native toolbar conventions.

## Scope

- **Show/hide guides** — eye icon (open/closed state) reflecting current visibility
- **Delete guides** — trash or clear icon

Icons must follow the project's stroke style: `fill="none"`, `stroke="currentColor"`.

## Files

- `src/components/ColumnGrid/GridPanel.tsx` — replace or augment existing controls
- `src/components/ColumnGrid/GridPanel.module.css` — icon sizing/alignment

## Acceptance Criteria

- [ ] Show/hide button displays eye-open icon when guides are visible, eye-closed when hidden
- [ ] Delete guides button displays a trash/clear icon
- [ ] Both icons use `currentColor` and respond to hover/active states
- [ ] Functionality unchanged (icons are additive to existing logic)
- [ ] Build passes with no errors
