# Issue 043: Margin-aware column and row guide generation

## Context

Currently columns and rows are generated relative to the full container (canvas or selection), ignoring any margin values. This issue makes columns and rows margin-aware: when margins are present, guides are distributed within the area defined by those margins.

## Behavior

### Columns (affected by Left + Right margins)

```
effectiveOffsetX = containerOffsetX + (left || 0)
effectiveWidth   = containerWidth - (left || 0) - (right || 0)
```

### Rows (affected by Top + Bottom margins)

```
effectiveOffsetY   = containerOffsetY + (top || 0)
effectiveHeight    = containerHeight - (top || 0) - (bottom || 0)
```

### Container Source
- Selection active → margins offset from selection bounds
- No selection → margins offset from canvas bounds

## Files

- `src/components/ColumnGrid/GridPanel.tsx`
- `src/services/gridGenerator.ts`

## Acceptance Criteria

- [ ] Columns respect left + right margins when present (any combination)
- [ ] Rows respect top + bottom margins when present (any combination)
- [ ] Single-side margin shifts column start without shrinking from the other side
- [ ] Width and Height computed fields reflect margin-adjusted real size
- [ ] No margins → behavior identical to current
- [ ] Works with both canvas and active selection as container
- [ ] Build passes with no errors
