# Issue 002: New icons for Margins, Columns, and Rows sections

## Context

The Margins, Columns, and Rows sections in the main panel currently lack dedicated icons or use inconsistent visual indicators. This issue adds a coherent icon to each section header to reinforce the section's purpose at a glance.

## Scope

Design and implement one SVG icon per section:
- **Margins** — represents offset from document edges
- **Columns** — represents vertical grid divisions
- **Rows** — represents horizontal grid divisions

Icons must follow the same stroke style as the Side Guides bar icons: `fill="none"`, `stroke="currentColor"`, strokeWidth 1.5–2.

## Files

- `src/components/ColumnGrid/GridPanel.tsx` — add icon to each section header
- `src/components/ColumnGrid/GridPanel.module.css` — style section header icons

## Acceptance Criteria

- [ ] Icon renders next to the section label for Margins, Columns, and Rows
- [ ] All icons use `currentColor` stroke (respond to theme/state)
- [ ] Icons are visually consistent in size and weight
- [ ] Build passes with no errors
