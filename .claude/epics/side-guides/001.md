# Install lucide-react

## Objetivo

Instalar `lucide-react` como dependencia de producción y verificar que webpack lo bundlea correctamente en UXP.

## Pasos

1. `npm install lucide-react`
2. Verificar que aparece en `dependencies` de `package.json` (no devDependencies)
3. Hacer un build de prueba: `npm run build` — debe compilar sin errores
4. Confirmar en `dist/index.js` que los imports de lucide se incluyen

## Acceptance Criteria

- [ ] `lucide-react` en `package.json` dependencies
- [ ] `npm run build` exitoso
- [ ] No warnings de módulos faltantes en webpack output
