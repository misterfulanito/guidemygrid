# Wire SideGuidesBar en GridPanel + bump v1.5.0

## Objetivo

Integrar `SideGuidesBar` en `GridPanel.tsx` y actualizar la versión del plugin a v1.5.0.

## Cambios en GridPanel.tsx

1. Importar el componente:
```typescript
import { SideGuidesBar } from '../SideGuidesBar/SideGuidesBar';
```

2. Insertar entre `contextBar` y `content`:
```tsx
{/* ── Context bar ── */}
<div className={styles.contextBar}>...</div>

{/* ── Side Guides Bar ── */}
<SideGuidesBar
  containerWidth={containerWidth}
  containerHeight={containerHeight}
  offsetX={offsetX}
  offsetY={offsetY}
  disabled={!document}
/>

{/* ── Content ── */}
<div className={styles.content}>...
```

## Bump de versión

### src/version.ts
```typescript
export const VERSION = '1.5.0';
```

### manifest.json
Actualizar campo `version` a `"1.5.0"`.
