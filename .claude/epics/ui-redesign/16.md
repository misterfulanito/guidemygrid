## Funciones a implementar

### photoshopBridge — nuevas operaciones
```typescript
// Alinear guías al documento o selección
alignGuides(direction: 'left' | 'centerH' | 'right' | 'top' | 'centerV' | 'bottom', target: 'document' | 'selection'): Promise<void>

// Toggle visibilidad de guías
toggleGuidesVisibility(visible: boolean): Promise<void>
getGuidesVisible(): Promise<boolean>

// Limpiar guías
clearGuides(scope: 'all' | 'plugin-only'): Promise<void>
```

### useUIStore — nuevo estado
```typescript
guidesVisible: boolean
setGuidesVisible: (v: boolean) => void
```

### App.tsx — conectar botones
- Toolbar align buttons → `photoshopBridge.alignGuides(...)`
- Clear button → `photoshopBridge.clearGuides('all')`
- Eye button → toggle `guidesVisible` + `photoshopBridge.toggleGuidesVisibility(...)`
