# Design Tokens — Dark Theme CSS Variables

## Contexto

El código actual usa variables de Adobe Spectrum (light theme). El diseño Pencil aprobado es dark theme con acento naranja. Necesitamos un sistema de tokens propio que conviva con UXP.

## Tokens a extraer del diseño Pencil

### Colores
- `--gmg-bg-panel`: fondo del panel (~#1E1E1E)
- `--gmg-bg-input`: fondo de inputs (~#2A2A2A)
- `--gmg-bg-toolbar`: fondo toolbar (~#1A1A1A)
- `--gmg-border`: bordes (~#3A3A3A)
- `--gmg-text-primary`: texto principal (~#F0F0F0)
- `--gmg-text-secondary`: labels (~#888888)
- `--gmg-text-placeholder`: placeholder (~#555555)
- `--gmg-accent`: naranja acento (#FF5700)
- `--gmg-accent-hover`: naranja hover (#FF6B1A)
- `--gmg-tab-active-border`: underline tab activo (naranja o blanco)

### Tipografía
- `--gmg-font-base`: familia de fuente (Adobe Clean, system-ui)
- `--gmg-font-size-sm`: 10px (labels)
- `--gmg-font-size-base`: 12px
- `--gmg-font-size-section`: 13px (títulos de sección)

### Espaciado
- `--gmg-section-padding`: padding interno de secciones
- `--gmg-gap-inputs`: gap entre inputs

## Deliverable
- Archivo `src/styles/tokens.css` con todas las variables
- Importado en `index.tsx` o `App.tsx`

## Acceptance Criteria
- [ ] Tokens extraídos del diseño Pencil (valores exactos, no aproximados)
- [ ] Archivo tokens.css creado e importado
- [ ] No hardcodear valores en los módulos CSS — solo usar variables
