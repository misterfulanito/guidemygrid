<objective>
Stand up the Gumroad distribution front-end without any scripting or duplicate binary hosting. Purpose: Give a first-time seller everything they need to publish the front-end and correctly wire the "always current" download link, then confirm the live link resolves before the phase closes.
Output: `gumroad-page-copy.md` (page copy + step-by-step instructions); a human-verify checkpoint confirming the live listing redirects to the current GitHub Release.
</objective>
